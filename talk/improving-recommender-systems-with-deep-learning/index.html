    Improving Recommender Systems with Deep Learning

    Video
    Speaking at the ReWork AI in Retail and Advertising Summit 2019

    Abstract

    In this talk I explain how Dataiku improved an online vacation retailer’s recommender system by leveraging pre-trained deep learning models to derive user preference information from images. This transfer learning approach can enable companies to use state-of-the-art machine learning methods without having deep learning expertise.
